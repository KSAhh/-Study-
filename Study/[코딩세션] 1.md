### 실습

- 작업 폴더내 → "Git BASH HEAR" 클릭 → `$ code .` → vscode 열림
- `python --version` : 파이썬 버전
- `pwd` : 현재 파일 위치
- `python -m venv myvenv` : 가상환경 폴더(myvenv) 생성
- `source myvenv/Scripts/activate` : 가상환경 ON
- `pip install django`: django설치. Python Install Package
