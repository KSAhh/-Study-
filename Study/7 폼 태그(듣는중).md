# form  
- `<form>`  
- 속성  
> `action`  
>> 해당 데이터를 처리할 웹서버 url을 담고 있음  
>> 데이터 보낼 url 지정  

> `method`  
>> 보내는 방식 지정  
- ```method="get"```, ```method="post"```  

### GET  
- 주소와 데이터를 함께 적어서 보냄 / ex) 엽서카드  
- 목적 : 데이터 조회 / ex) 검색기능  
- ★[GET] [GET2]  


### POST  
- 주소와 데이터URL을 body부분에 숨겨서 보냄 / ex) 편지  
- 목적 : 서버에 있는 데이터를 쓰기, 수정, 삭제  / ex) 게시물 작성   
- ★ [POST] [POST2]
