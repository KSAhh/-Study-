# PK [8] 
- Primary Key  
- object를 식별할 수 있는 값 / object가 가져야하는 식별자  
- 중복(x)  
- model을 만들때, pk를 정해주지 않으면 id를 만들어줌[7]  
- 사용법  
```python
    class MyModel(models.Model):
        my_pk = models.IntegerField(primary_key=True)
```
